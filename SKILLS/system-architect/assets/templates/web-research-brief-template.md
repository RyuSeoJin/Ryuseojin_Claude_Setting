# 웹 리서치 브리프 템플릿

> web-researcher 에이전트가 리서치 완료 후 작성하는 표준 산출물 형식.
> WebSearch/WebFetch 도구를 사용한 자동 수집 결과를 구조화한다.

---

## 신뢰도 등급 기준

| 등급 | 설명 | 예시 |
|------|------|------|
| A | 공식 소스 (개발사 문서, 패치노트) | 공식 홈페이지, 개발자 블로그 |
| B | 공인 커뮤니티 위키 (검증된 데이터) | Fandom Wiki, 나무위키 (검증 항목) |
| C | 커뮤니티 합의 (다수 유저 확인) | Reddit 다수 동의, 커뮤니티 검증 |
| D | 개별 유저 보고 (미검증) | 개인 블로그, 유튜브 단일 영상 |
| E | 추정/외삽 (직접 데이터 부재) | 매출 추정, 비공개 수치 역산 |
